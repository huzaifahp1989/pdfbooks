# Books Folder

Place your PDF files in this folder.

## File Naming Convention

Use descriptive, lowercase names with hyphens:
- ✅ Good: `fortress-of-the-muslim.pdf`
- ✅ Good: `complete-guide-to-prayer.pdf`
- ❌ Avoid: `Book1.pdf`, `file (1).pdf`

## Supported Formats

- PDF (.pdf) - Primary format
- Other formats can be supported with additional code modifications

## Adding Books

1. Copy your PDF file to this folder
2. Note the exact filename
3. Add the book entry to the `booksDatabase` array in `script.js`
4. Use the filename in the `pdfUrl` field: `books/your-filename.pdf`
